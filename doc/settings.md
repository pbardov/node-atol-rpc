# Настройки подключения (Settings)

## Тип Settings

Библиотека принимает объект настроек подключения при создании экземпляра `AtolRpc`.

Поля:

- `host: string` — хост RPC-сервиса (обычно `127.0.0.1`, если ДТО запущено локально).
- `port: number` — порт RPC-сервиса (часто `50000`).
- `timeout?: number` — таймаут выполнения одного RPC-вызова в миллисекундах.

Пример:

```ts
import AtolRpc, { type Settings } from 'node-atol-rpc';

const settings: Settings = {
  host: '127.0.0.1',
  port: 50000,
  timeout: 10_000,
};

const atol = new AtolRpc(settings);
```

## Практические рекомендации

- Увеличивайте `timeout` для операций, которые могут выполняться долго (печать больших чеков, обмен с ОФД, операции с маркировкой).
- Если RPC-сервис недоступен, проверяйте:
  - что ДТО/драйвер АТОЛ запущен и RPC включён;
  - настройки firewall;
  - правильность `host`/`port`.
